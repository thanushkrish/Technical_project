### FOR CREATING INFRASTRUCTURE ###

Fork the code in your local machine
Install terraform 
Install AWS CLI 
Use "aws configure" command with Access Key and secret key for allowing terraform to create resources
create a directory and initiate terraform
make sure all the files with .tf are in that directory 
run "terraform plan" command and if the plan is sucess prcoeed with "terraform apply" command to start bulidng the resources



### FOR CI/CD Pipleline ####

Install Git and jenkins in your local machine
Configure GitHub and DockerHub credentials for the job to run.
Install maven and unzip or untar the package 
configure pom.xml for building the code into a war file.
Make sure to mention the right middleware in Dockerfile
install maven plugin and configure the path (home directory of maven) 
create a pipeline job using Jenkinsfile and run the Job.
